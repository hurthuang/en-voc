# 英文單字測驗

線上英文單字／點字（Braille）測驗工具。線上使用：**https://hurthuang.github.io/en-voc/**

涵蓋國中（康軒／翰林／南一）與國小（康軒／翰林／何嘉仁）各版本教材單字，支援語音朗讀、拼字測驗、UEB 點字輸入練習。

---

## 快速開始

1. 打開 https://hurthuang.github.io/en-voc/
2. 選「教育階段」（國中／國小），再依序選版本、年級/系列、學期/冊次、課次
3. 按「選擇」載入題庫，開始作答

---

## 選題目

- **國中**：版本（康軒／翰林／南一）→ 年級（7/8/9）→ 學期（上/下）→ 課次
- **國小**：版本（康軒／翰林／何嘉仁）→ 系列（如 Wonder World、HereWeGo、Super Fun）→ 冊次 → 課次／主題
- 也可以自己上傳單字檔（`.txt`，每行「英文\t中文\t點字」用 Tab 分隔），不受限於內建題庫

---

## 顯示模式

答題區上方有三個單選鈕，決定畫面上顯示多少英文提示：

| 模式 | 效果 |
|---|---|
| 顯示英文 | 完整顯示英文單字 |
| 顯示首尾 | 只顯示每個字的第一個和最後一個字母，中間用 `-` 遮住（例如 apple → a---e） |
| 不顯示英文 | 完全不顯示，只看中文意思作答 |

不管哪個模式，答案判定都是比對完整英文拼字，顯示模式只影響提示的多寡。

---

## 點字（Braille）輸入模式 — 同時也是 UEB 縮寫練習

按 **Ctrl + B** 切換「一般輸入模式」／「點字輸入模式」。

點字模式下，正確答案是這個單字/句子的 **UEB（Unified English Braille）二級縮寫點字碼**，不是逐字母拼出來的點字——也就是說，只要在這個模式下練習，同時就是在練習 UEB 縮寫規則本身（例如 "you" 要打成縮寫用的單一格 `⠽`，而不是逐字母拼出 y-o-u）。

點字用標準六點鍵盤輸入（比照點字機按鍵配置）：

| 按鍵 | 對應點位 |
|---|---|
| S | 點1 |
| D | 點2 |
| F | 點3 |
| J | 點4 |
| K | 點5 |
| L | 點6 |

同時按下對應的按鍵組合，放開後就會輸出那一格點字。其他點字模式按鍵：

| 按鍵 | 功能 |
|---|---|
| 空白鍵 | 輸入一個空白格（⠀） |
| A | 刪除前一格 |
| ; （分號） | 換行並送出答案（等同按「送出」） |

不管在一般模式或點字模式，**Enter** 都可以送出答案、**Esc** 可以清空目前的作答欄。

## 語音與其他設定

- **唸中文** / **唸拼字**：作答時是否自動朗讀中文意思、是否逐字母拼讀英文
- **測驗模式**：把目前題庫固定成一輪（不會無限重複出題），會顯示已答題數與計時，適合正式測驗使用；不勾選則是自由練習模式，會一直重複出題
- **語音速度**：拖曳調整朗讀速度
- **題目字體大小**：拖曳調整畫面文字大小
- **重複發音 [Ctrl+Q]**：再唸一次目前題目

畫面右上角的 🌙／☀️ 小圖示可以切換深色／淺色主題。
