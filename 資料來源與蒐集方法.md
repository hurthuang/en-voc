# 英文單字表蒐集方法與經驗記錄

記錄如何從四大出版社（康軒／翰林／南一／何嘉仁）的教師數位平台找到並擷取「各課單字表」，供後續補做／更新使用。

---

## 通用策略

1. 教科書出版社的教師平台通常都有「課本PDF」書末的**逐冊附錄**，裡面才是真正完整的「字詞例句表／Word Bank／各課生詞表」——這是首選來源（比書中每頁的單字盒或閃卡更完整，含詞性、中譯、例句）。
2. **陷阱**：很多平台另外有一個叫「附錄本PDF」的獨立資源分類，聽起來像單字附錄，但實際上常常是**測驗題本**（綜合測驗/閱讀測驗），不是單字表。真正的單字附錄通常在「課本PDF」分類自己的檔案列表最後面（檔名常帶「後附錄」字樣）。每次都要點開檔案內容確認，不要只看檔名分類。
3. **陷阱**：帳號的學年度／學期存取權限會隨時間變化。曾經判定「此帳號只能存取下學期」，之後重新檢查卻發現上學期權限已開通。不要只靠先前一次的結論，需要時應重新檢查學年度下拉選單的完整選項。
4. 自然發音（phonics）導向的入門教材（如康軒 Super Starter／Bravo ABC／Cool ABC、翰林 Hooray／Twinkle、何嘉仁 WOW! STORY／Fun World 第一冊）通常**沒有主題式單字表**，只有逐課字母教學＋單字卡圖片，難以批次蒐集，建議直接跳過並記錄原因。

---

## 文字擷取技術

### 技術一：PDF.js 直接讀取（用於線上PDF預覽viewer，康軒/翰林/南一皆適用）

在瀏覽器開啟PDF預覽頁後，於該分頁執行 JS（透過 Playwright `browser_evaluate`）：

```js
async () => {
  const app = window.PDFViewerApplication;
  await app.pdfLoadingTask.promise;
  const doc = app.pdfDocument;
  const out = [];
  for (let i = 1; i <= doc.numPages; i++) {
    const page = await doc.getPage(i);
    const tc = await page.getTextContent();
    const text = tc.items.map(it => it.str).join(' ');
    out.push('--PAGE ' + i + '--\n' + text);
  }
  return out.join('\n\n');
}
```

- 若PDF超過約20頁，輸出會超過工具的token上限，要分批擷取（例如先1~16頁，再17~最後），先用一個小函式讀取 `doc.numPages` 決定要分幾批。
- 這個方法只對「原生數位文字PDF」有效，若PDF是掃描影像（如何嘉仁大部分教材），文字內容會是空的。

### 技術二：pdftotext（本機下載的PDF檔案）

若平台是直接下載檔案（而非開瀏覽器內建viewer預覽），下載後用：

```bash
pdftotext -layout "檔名.pdf" 輸出.txt
```

- Windows 環境下 Git Bash 內建 `/mingw64/bin/pdftotext`（poppler套件的一部分），不需額外安裝。
- 同樣只對原生文字PDF有效；若輸出檔案幾乎是空的（幾個bytes），代表是掃描影像PDF，此法失效。
- 中文檔名在bash中容易亂碼，建議下載後先用 `mv 原檔名* 新ASCII檔名.pdf` 重新命名再處理。

### 技術三：antiword（舊版 .doc 檔案，非.docx）

翰林 Dino 系列的「中譯表.doc」是舊版 OLE 複合文件格式（非ZIP格式的.docx），用：

```bash
antiword -m UTF-8 "檔名.doc"
```

`/mingw64/bin/antiword` 在此環境已內建可用，輸出乾淨UTF-8文字含表格內容。

### 技術限制：掃描影像PDF（何嘉仁遇到的問題）

何嘉仁教師網（neweteaching.hess.com.tw）下載的課本／備課用書PDF幾乎都是掃描影像檔，沒有文字圖層，`pdftotext`擷取結果是空的。此環境沒有 `tesseract`／`pdftoppm`／OCR相關工具，無法自動化擷取，只能靠：
- 逐頁截圖＋人工視覺判讀（慢，但可行）
- 或尋找該平台是否有其他「原生文字」格式的資源（如學用電子書 bookonline.hess.com.tw，但需要另外註冊帳號，本次未能驗證其內容格式）

## 各出版社平台細節

### 康軒（digitalmaster.knsh.com.tw，「康軒數位高手」）

- URL hash 帶學年度參數 `year=NNNS`（NNN=學年度，S=1上/2下），如 `year=1151` = 115學年度上學期。
- **切換學年度必須點擊畫面上的年度下拉按鈕**（例如「114(下) ▼」），選單彈出後點選目標學年度；單純改URL參數不會讓SPA真的切換資料。
- 資源分類：點「單檔下載」開啟modal → 有「教用／學用」分頁，點「教用」展開檔案列表 → 找目標檔案的「預覽」按鈕，會開新分頁用內建PDF.js viewer開啟。
- **切換到下一個資源分類前，一定要先點modal的「關閉」按鈕**，否則點擊可能被前一個modal攔截。
- 國中英語：真正的單字表檔名格式如「X上/下課本_後附錄N-字詞例句表(教).pdf」（N視冊次而異，通常是1或2或3，需要看檔案列表確認），在「英語X上/下課本PDF」分類裡，不是在「英語X上/下附錄本PDF」（那是測驗題本）。
- 國小英語：單字不在書末獨立附錄，而是每個Unit自己的PDF內固定在「Vocabulary」頁；部分冊次另有「全冊合訂」PDF書末含逐課中英對照＋生字索引，資料較完整，優先採用。

### 翰林（edisc3.hle.com.tw「行動大師3」／GWTY頁面，hanlindigi.hle.com.tw查詢用）

- GWTY資源頁可用JS API直接切換條件（比UI點擊更快）：
  ```js
  window.app.GWTYPageV2023.instance.lstChooseCategory_onChange('英語','<product code>')
  window.app.GWTYPageV2023.instance.lstChooseYear_onChange('<year>')
  ```
- 國小產品代碼：EEC=HereWeGo、EEB=Hooray、EE2=Dino、EEF=Twinkle；國中統一代碼 JEN（不分系列）。
- 國中冊次對應：iE1/iE3/iE5＝英語1/3/5＝7/8/9年級**上**學期；iE2/iE4/iE6＝英語2/4/6＝7/8/9年級**下**學期（奇數=上，偶數=下）。
- 單字來源：「課本PDF」→「後附錄N-Word Bank-教用.pdf」（多數N=2，英語6/iE6因多一個彈性單元附錄而變成N=3，需視實際檔案列表確認）。
- 國小 HereWeGo 的「Words I Know」單字表是**累加式**：每一冊檔案都包含「第一冊到該冊」的全部詞彙，按**主題分類**（非按課次），格式是.docx（用PowerShell內建.NET ZipFile解壓縮讀取，因為.docx本質是ZIP壓縮檔）。
- 國小 Dino 系列的「中譯表.docx／.doc」內容分「應用字彙／認識字彙／音韻字彙」三類；部分冊次是舊版.doc格式需用antiword（見上）。
- 帳號可能同時擁有不同學年度的存取權，例如翰林國中帳號同時有115上（給iE1/3/5用）與114下（給iE2/4/6用），需要切換學年度下拉選單分別擷取。

### 南一（nanibox3.oneclass.com.tw「NaniBox3」）

- 注意：這個服務可能已經開在瀏覽器分頁但沒被識別出來，可從頁尾版權文字「NAN I BOOK ENTERPRISE CO., LTD」確認是南一書局平台。另有「NaniBox2.0」(onebox2.oneclass.com.tw/fileDownload) 是舊版/另一入口。
- UI是一串下拉選單（學制/科目/學年度/年級冊次），每個都要點開才會出現 `[role="option"]` 選項列表；卡住的下拉選單/modal可按 Escape 關閉。
- 國小：截至本次調查，「科目」下拉選單中**沒有英語選項**（只有國語/數學/生活/自然/社會/健體/綜合/藝文），南一國小英語的存取路徑仍未確認，可能需要不同帳號權限或其他入口。
- 國中：左側導覽「備課素材」→「01課習備PDF」→「課本PDF」自己的「附錄」檔案（不是同層的「附錄本PDF」，那是測驗題本），單一檔名格式如「X上/下課本PDF_附錄_教用.pdf」（整冊合一，不像康軒/翰林拆課次分檔）。
- 內容含「各課生詞表」＋「Word Power」主題延伸字＋「Idioms and Expressions」慣用語，皆需一併蒐集納入對應課次。
- 特殊案例：某些檔名看似正式課次（如「3下課本PDF_L5/L6」）但打開後其實是「PROJECT 1/2」等彈性使用延伸活動，非正式編號課次，沒有生詞表——遇到課次數量對不上時，要直接開檔案確認內容，不要單憑檔名判斷。

### 何嘉仁（neweteaching.hess.com.tw「何嘉仁國小英語教師網」）

- 只有國小英語教材，無國中系列。5大系列：WOW! STORY、Fun World、Go! Go! World、Go Magic!、Super Fun。
- 「教學資源」頁用 學年／系列／冊別／服務 四層下拉篩選；選好篩選條件後要按「查詢」按鈕（注意是`<input type="button" value="查詢">`，不是一般`<button>`元素，用文字內容找不到，要用`input[value="查詢"]`）畫面才會更新。
- **課本／備課用書PDF幾乎都是掃描影像檔**，無文字圖層，`pdftotext`完全擷取不到內容（WOW! STORY／Fun World／Go Magic!／Go! Go! World 這4系列都是如此，屬於自然發音字母教材，本來就沒有主題式單字表）。
- **後來找到真正可用的來源**：Super Fun 系列的「教學資源」清單裡有一份「單元字彙/課文中譯表.docx」，是逐課的文字版中譯表，且不需登入即可直接下載檔案本身。第1～8冊皆有（單數冊要選學年度「114上」、雙數冊要選「114下」才會出現在冊別選單），第9/10冊確認沒有這份檔案。詳見 `英文單字表_何嘉仁.md`。
